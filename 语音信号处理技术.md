## 二、实验原理

### **1. FFT（Fast Fourier Transform，快速傅里叶变换）**

FFT 是 **DFT（离散傅里叶变换）** 的高效计算算法，用于将时域信号转换到 **频域**，分析其频率成分。
 DFT 公式如下：
$$
X(k) = \sum_{n=0}^{N-1} x(n) e^{-j 2\pi kn/N}, \quad k = 0, 1, ..., N-1
$$
FFT 通过分治法优化计算，使时间复杂度降为O(NlogN)。

- 语音信号的 **频谱分析**（如共振峰、基音分析）。

- 语音 **滤波**（低通、高通、带通）。

- **语音识别、压缩**（如 MFCC 特征提取）。

  **特点:**

- 计算速度快，适合大规模数据。
- 适用于稳态信号（周期性频率分析）。
- 不能分析非平稳信号的 **时变特性**（频率随时间变化的情况）。
- 频率分辨率受采样窗口影响。

------

### **2. DWT（Discrete Wavelet Transform，离散小波变换）**

DWT 通过多尺度分析（Multi-Resolution Analysis, MRA），将信号分解为 **低频部分（近似系数）** 和 **高频部分（细节系数）**，然后继续递归分解低频部分。

小波变换的基本公式：
$$
X(a, b) = \sum_{n} x(n) \psi_{a,b}(n)
$$
其中 **小波基函数** 
$$
\psi_{a,b}(n)
$$
具有时频局部化特性。

- **语音降噪**（滤除噪声，保留语音信息）。

- **特征提取**（如基于小波包分解的语音识别）。

- **非平稳信号分析**（适用于时变频率成分）。

  **特点：**

- 适用于非平稳信号，可同时分析 **时域** 和 **频域信息**。
- 适用于压缩、去噪等应用。
- 小波基的选择较复杂（如 Daubechies、Haar）。
- 计算复杂度比 FFT 略高。

------

### **3. DCT（Discrete Cosine Transform，离散余弦变换）**

DCT 也是一种频域变换，与 FFT 类似，但只使用 **余弦基**，适用于 **能量集中度高** 的信号处理，通常用于信号的 **压缩** 和 **特征提取**。

DCT 公式：
$$
X(k) = \sum_{n=0}^{N-1} x(n) \cos\left[\frac{\pi k(2n+1)}{2N}\right]
$$
它将信号变换到 **频域**，但不会引入复数运算。

- **语音压缩**（如 MP3、AAC）。

- **MFCC（梅尔频率倒谱系数）特征提取**，用于 **语音识别**。

- **语音增强**（去噪、回声消除）。

  **特点：**

- 变换后能量集中（低频分量较多），适合压缩和降噪。
- 没有复数运算，计算复杂度较低。
- 适用于平稳信号，不能很好分析时变特性。
- 对某些类型的噪声去除能力不如 DWT。

------



## 三、实验过程

### 1. FFT（快速傅里叶变换）

```matlab
clc;
clear all;
close all;

[x, fs] = audioread('./test.wav'); % 读取音频文件
fx = fft(x); % 对音频信号执行 FFT

subplot(2, 1, 1);
plot(x);
title('Original Signal'); % 显示原始信号

subplot(2, 1, 2);
plot(abs(fftshift(fx))); % fftshift 函数用于将频谱居中
title('Frequency Spectrum');

saveas(gcf, './pic/FFT.png'); % 保存图像

```

1. **读取音频文件**：
    `audioread` 读取音频文件 `test.wav`，`x` 是音频信号，`fs` 是采样率。
2. **FFT 变换**：
    `fft(x)` 计算信号的频谱信息，将时域信号转换到频域。
3. **绘图：**
   - 第一幅图：**原始信号** `x`（时域）。
   - 第二幅图：`fftshift` 处理后的 **频谱**，`abs(fx)` 取模值，去掉复数部分。
4. **保存结果**：
    `saveas(gcf, './pic/FFT.png')` 将频谱图像保存。

#### **功能**

- **分析音频信号的频率成分**。
- **可用于滤波或特征提取**。

### 2. DWT（离散小波变换）

#### (1) 单级小波变换（dwt）

```matlab
clc;
clear all;
close all;

[a, fs] = audioread('./test.wav'); % 读取音频文件
plot(a)

[ca1, cd1] = dwt(a(:, 1), 'db4'); % 使用 db4 小波进行单级 DWT
a0 = idwt(ca1, cd1, 'db4', length(a(:, 1))); % 逆变换，重构信号

subplot(4, 1, 1);
plot(a(:, 1));
title('Original Signal');

subplot(4, 1, 2);
plot(cd1);
title('Detail Coefficients'); % 细节系数（高频部分）

subplot(4, 1, 3);
plot(ca1);
title('Approximation Coefficients'); % 近似系数（低频部分）

subplot(4, 1, 4);
plot(a0);
title('Reconstructed Signal'); % 重构后的信号

saveas(gcf, './pic/DWT.png'); % 保存图像

```

1. DWT 变换：

   ```
   dwt(a(:,1), 'db4')
   ```

    进行 **单级离散小波变换**，使用 Daubechies 4（db4）小波：

   `ca1`：**近似系数**，表示低频部分（信号主要成分）。

   `cd1`：**细节系数**，表示高频部分（瞬时变化、噪声）。

2. IDWT 逆变换：

   `idwt(ca1, cd1, 'db4', length(a(:,1)))` 通过逆变换 **重构原始信号**。

3. 绘图：

   - 第一幅图：原始音频信号。
   - 第二幅图：高频 **细节系数（cd1）**。
   - 第三幅图：低频 **近似系数（ca1）**。
   - 第四幅图：**重构信号（a0）**，应该与原信号接近。

##### **功能**

- DWT 适用于时变信号，可用于：
  - **语音降噪**（高频部分通常是噪声）。
  - **语音特征提取**（识别低频特征）。
  - **压缩编码**（去除不重要的高频分量）。

### （2）一级小波分解（wavedec）

```matlab
clc;
clear all;
close all;

[a, fs] = audioread('./test.wav'); % 读取音频文件
plot(a)
title('Original Signal')

[ca1, cd1] = wavedec(a(:, 1), 1,'db4'); % 1 级小波分解
a0 = waverec(ca1, cd1, 'db4'); % 重构信号

subplot(4, 1, 1);
plot(a(:, 1));
title('Original Signal')

subplot(4, 1, 2);
plot(cd1);
title('Detail Coefficients') % 细节系数（高频部分）

subplot(4, 1, 3);
plot(ca1);
title('Approximation Coefficients') % 近似系数（低频部分）

subplot(4, 1, 4);
plot(a0);
title('Reconstructed Signal') % 重构后的信号

saveas(gcf, './pic/wavedec.png'); % 保存图像

```

1. 读取音频：

   `audioread('./test.wav')` 读取音频文件，`a` 是音频信号，`fs` 是采样率。

2. 小波分解（1 级）：

   `wavedec(a(:, 1), 1, 'db4')`

    进行 1 级小波分解，使用 Daubechies 4（db4） 小波：

   `ca1`（**近似系数**）：表示低频部分，包含主要信号特征。

   `cd1`（**细节系数**）：表示高频部分，包含突变信息或噪声。

3. 小波重构：

   `waverec(ca1, cd1, 'db4')` 通过 **逆小波变换** 重建信号。

4. 绘图：

   - **原始信号**（时域）。
   - **细节系数（高频部分）**，可能包含高频噪声。
   - **近似系数（低频部分）**，主要是信号主体。
   - **重构信号**，应接近原始信号。

##### **功能**

- **可用于信号降噪**（去掉高频 `cd1`）。
- **特征提取**（分析 `ca1` 和 `cd1`）。

#### (3)三级小波分解

```matlab
clc;
clear all;
close all;

[a, fs] = audioread('./test.wav'); % 读取音频文件
plot(a)

[c,l]=wavedec(a(:,2),3 ,'db4'); % 3 级小波分解
ca3=appcoef(c,l,'db4',3); % 提取 3 级近似系数
cd3=detcoef(c,l,3); % 提取 3 级细节系数
cd2=detcoef(c,l,2); % 提取 2 级细节系数
cd1=detcoef(c,l,1); % 提取 1 级细节系数
a0=waverec(c,l,'db4'); % 重构信号

subplot(6, 1, 1);
plot(a(:, 2));
title('Original Signal');

subplot(6, 1, 2);
plot(cd1);
title('Detail Coefficient 1'); % 1 级细节系数（最高频部分）

subplot(6, 1, 3);
plot(cd2);
title('Detail Coefficient 2'); % 2 级细节系数（次高频部分）

subplot(6, 1, 4);
plot(cd3);
title('Detail Coefficient 3'); % 3 级细节系数（中等频率）

subplot(6, 1, 5);
plot(ca3);
title('Approximation Coefficient 3'); % 3 级近似系数（最低频部分）

subplot(6, 1, 6);
plot(a0);
title('Reconstructed Signal'); % 重构后的信号

saveas(gcf, './pic/wavedec_3.png');

```

1. 小波分解（3 级）：

   `wavedec(a(:,2), 3, 'db4')`

   进行 3 级小波分解：

   `ca3`：**第 3 级近似系数**（最低频）。

   `cd3`：**第 3 级细节系数**（中等频率）。

   `cd2`：**第 2 级细节系数**（次高频）。

   `cd1`：**第 1 级细节系数**（最高频）。

2. 小波重构：

   `waverec(c,l,'db4')` 通过 **逆小波变换** 重建信号。

3. 绘图：

   - **原始信号**（时域）。
   - **第 1 级细节系数**（最高频噪声部分）。
   - **第 2 级细节系数**（次高频）。
   - **第 3 级细节系数**（中频）。
   - **第 3 级近似系数**（主要信号特征）。
   - **重构信号**。

##### **功能**

- **多尺度信号分析**（分解不同频率成分）。
- **信号压缩**（可以丢弃高频部分）。
- **降噪**（去掉 `cd1`，保留 `ca3`）。

### 3. DCT（离散余弦变换）

```matlab
clc;
clear all;
close all;

[a, fs] = audioread('./test.wav'); % 读取音频文件
plot(a)
title('Original Signal');

a1 = dct(a(:, 1)); % 计算 DCT 变换
a0 = idct(a1); % 计算逆 DCT

subplot(3, 1, 1);
plot(a(:, 1));
title('Original Signal');

subplot(3, 1, 2);
plot(a1);
title('DCT Coefficients'); % DCT 系数

subplot(3, 1, 3);
plot(a0);
title('Reconstructed Signal'); % 逆变换信号

saveas(gcf, './pic/DCT.png'); % 保存图像

```

1. DCT 变换：

   `dct(a(:,1))` 计算 **离散余弦变换**，将时域信号转换为 **频域表示**，但只包含余弦分量。

   DCT 具有 **能量集中性**，前几个系数通常包含信号的主要信息。

2. IDCT 逆变换：

   `idct(a1)` 计算 **逆 DCT**，将频域信号转换回时域。

3. 绘图：

   - 第一幅图：**原始信号**。
   - 第二幅图：**DCT 系数**（*通常前几个值较大*）。
   - 第三幅图：**重构信号**（应接近原信号）。

#### **功能**

- DCT 主要用于语音压缩：
  - **语音编码**（如 MP3、AAC）。
  - **特征提取（如 MFCC）**。
  - **去噪、回声消除**。

## **四、总结对比**

| 方法    | 主要用途                           | 适用信号       | 优势                   | 劣势             | 应用                              |
| ------- | ---------------------------------- | -------------- | ---------------------- | ---------------- | --------------------------------- |
| **FFT** | 频谱分析、滤波、语音识别           | **平稳信号**   | 快速计算频谱           | 不适用于时变信号 | **语音频谱**                      |
| **DWT** | 语音降噪、特征提取、非平稳信号分析 | **非平稳信号** | 适用于时变信号         | 计算较复杂       | **语音降噪、时频分析**            |
| **DCT** | 语音压缩、特征提取（如 MFCC）      | **平稳信号**   | 能量集中度高，适合压缩 | 不适用于时变信号 | **语音压缩、特征提取（如 MFCC）** |
