## 一、实验要求

1. 生成：将南开校徽二值图加密，即秘密图像
2. 嵌入：利用奇偶校验位隐藏法，实现将秘密图像嵌入到位图中
3. 提取：将秘密图像提取出来
4. 解密：将提取的秘密图像复原为南开校徽。

## 二、实验原理

将载体图像划分为若干**不重叠区域**，每个区域嵌入一位秘密信息。每个区域中多个像素的**最低有效位（LSB）**共同决定该区域的奇偶校验位。通过控制该奇偶校验值与秘密信息比特一致，即可完成嵌入。

### 方法一：**全部反转型嵌入**

#### 嵌入原理：

- 将图像划分为 n 个区域 I_1, I_2, ..., I_n

- 每个区域 I_i 中的 LSB 做奇偶校验：
  $$
  b_i = \sum_{j \in I_i} LSB(c_j) \mod 2
  $$

- 若 b_i = m_i，则无需修改；

- 若 b_i != m_i，则将**该区域所有像素的 LSB 取反**，使得新的校验值 b_i' = m_i

#### 提取原理：

- 接收方按相同方式划分区域；
- 计算每个区域的奇偶校验值 b_i，依次组合重构出秘密图像的二值信息。

------

### 方法二：**单点修改型嵌入**

#### 嵌入原理：

- 与方法一一致划分区域、计算奇偶位；
- 若 b_i != m_i，只需**随机翻转该区域中一个像素的 LSB**，即可改变奇偶性，使 b_i = m_i ；
- 优点是**改动更小、隐蔽性更强**，但可能不如全部反转型鲁棒。

#### 提取原理：

- 与方法一相同，只需重新计算每个区域 LSB 的奇偶校验值，即可还原秘密信息。



## 三、实验过程

> 本实验选择**单点修改型嵌入**

### 1. 生成阶段：将南开校徽二值图加密，即秘密图像

- **原图像（载体）**和**秘密图像**首先都转为灰度图（`rgb2gray`）；
- 对秘密图像进行**二值化**（`imbinarize`），以便只使用 0 和 1 来表示黑白信息。

#### **加密部分在原始二值图像基础上进行 XOR 加密**

```matlab
secret_bin = imbinarize(secret_gray);  % 转为二值图像
key = randi([0, 1], size(secret_bin));  % 随机生成0/1密钥矩阵
save('secret_key.mat', 'key');          % 保存密钥以便解密使用

secret_enc = xor(secret_bin, key);      % 加密：与密钥异或
figure; imshow(secret_enc); title('Encrypted Binary Secret Image');
saveas(gcf, './pic7/secret_encrypted.png');
```

#### 嵌入部分使用 `secret_enc` 进行隐藏

```matlab
code[m, n] = size(secret_enc);
...
stego_img = Hide(cover_gray, m, n, secret_enc);
```

#### **解密部分：添加 XOR 解密操作**

```matlab
load('secret_key.mat', 'key');           % 加载保存的密钥
secret_dec = xor(extracted_bin, key);    % 异或解密
extracted_logo = uint8(secret_dec * 255);  % 恢复为灰度图
figure; imshow(extracted_logo); title('Decrypted Secret Image');
imwrite(extracted_logo, './pic7/Extracted_Nankai.bmp');
saveas(gcf, './pic7/decrypted_secret.png');
```

------

### 2. 嵌入阶段：利用奇偶校验位隐藏法，实现将秘密图像嵌入到位图中

- 将原图像划分为不重叠的 `2×2` 像素块；
- 每个像素块中提取四个像素的**最低有效位（LSB）**，即第1位；
- 对这4个位做**奇偶校验**（即求和后对2取余）：
  - 如果校验值 = 秘密图像对应位，则跳过；
  - 如果不等，则**随机翻转**该块中某一个像素的 LSB，使校验值与秘密位一致。

>  这样一个像素块就可嵌入1位秘密信息，共需原图是秘密图的2倍大小。

```matlab
stego_img = Hide(cover_gray, m, n, secret_bin);
```

调用 Hide() 函数进行嵌入处理。该函数将载体图像划分为 m × n 个不重叠的 2×2 区域，每个区域对应秘密图像的一个比特。

嵌入原理：

1. 用 checksum() 函数计算每个 2×2 区域的最低有效位（LSB）的奇偶校验和。

2. 如果这个奇偶位与秘密图对应位置的比特不同，就随机修改该区域中一个像素的最低位，使其变为匹配的奇偶性。


这确保了修改最少像素（仅一个）的同时隐藏了一位秘密信息。

#### 校验和函数（奇偶性判断）

```matlab
function result = checksum(x, i, j)
    bits = [
        bitget(x(2*i-1, 2*j-1), 1),
        bitget(x(2*i-1, 2*j), 1),
        bitget(x(2*i, 2*j-1), 1),
        bitget(x(2*i, 2*j), 1)
    ];
    result = mod(sum(bits), 2);
end
```

`bitget(..., 1)` 取得像素的最低有效位，4 个像素求和后对 2 取模，即得该区域 LSB 的奇偶校验位。结果为 0 表示偶数个 1，为 1 表示奇数个 1。

------

### 3. 提取阶段：将秘密图像提取出来

- 对嵌入图像进行 `2×2` 分块；
- 提取每块4个像素的 LSB；
- 重新计算其**奇偶校验值**，还原为二值图像中的一个像素值（0 或 1）；
- 最终重构出隐藏的秘密图像的二值版本。

```matlab
extracted_bin = Extract(stego_img);
```

提取函数与嵌入思路一致：

- 用同样的方式将载体图划分成 2×2 区域；
- 对每一区域用 `checksum()` 得到 LSB 奇偶性；
- 将这些奇偶性结果依次还原为秘密图像。

提取结果仍为二值图像（logical 类型），不需要知道嵌入时修改的是哪一个像素。

------

### 4. 解密阶段：将提取的秘密图像复原为南开校徽

- 将提取得到的二值图像转换为逻辑图像或黑白图；
- 在显示或保存过程中，重新映射为黑白图（可选灰度填充）。

```matlab
extracted_logo = uint8(extracted_bin * 255);
```

将提取得到的二值图转换为标准灰度图（0/255），使其更接近原始校徽图像。此步骤相当于简单的图像格式还原，不涉及复杂解密算法。

## 四、实验总结

图像隐写术是一种信息隐藏技术，它通过将秘密信息嵌入到载体图像中，从而实现信息的隐蔽传输。本实验采用了奇偶校验位隐藏法，将南开校徽作为秘密图像嵌入到载体图像中，并实现了秘密信息的加密、嵌入、提取和解密过程。通过本实验，深入理解了图像隐写术的原理和应用，同时也加深了对图像加密、图像处理和数据恢复技术的掌握。
