## 一、实验要求

1. 隐藏：利用二值图像隐藏法，实现将秘密信息（可以是图像、文字等信息）嵌入到位图中；
2. 提取：将秘密信息提取出来。



## 二、实验原理

本实验基于一种简化的 Zhao-Koch 图像隐写算法，通过对图像划分为若干个小块（每块由连续 4 个像素构成的 1×4 区域），根据每个块中黑色像素的数量来嵌入二进制秘密信息。其核心思想是利用每个小块中**黑色像素点数量的奇偶性和数量特征**来表示比特 `0` 或 `1`，而修改的规则遵循“**最小修改，视觉无感知**”的原则。

------

#### 图像划分与编码规则

图像被划分为多个 1×4 的矩形像素块，对每个块统计黑色像素个数（值为 0 的像素），根据如下规则判断是否嵌入比特，以及嵌入什么比特：

| 黑像素个数 | 像素分布 | 含义   | 是否可用 |
| ---------- | -------- | ------ | -------- |
| 0          | 全白     | 无效块 | 否       |
| 1          | 1黑3白   | 表示 1 | 是       |
| 2          | 2黑2白   | 非法   | 否       |
| 3          | 3黑1白   | 表示 0 | 是       |
| 4          | 全黑     | 无效块 | 否       |

------

#### 信息嵌入规则

**嵌入 0：**

- 将当前块中黑色像素点数调整为 **3 个**。
- 若已有 3 个黑像素，无需修改；
- 若为 1、2 或 4 个黑像素，则**尽量保持原有黑像素位置不变**，仅最少翻转必要的白像素为黑像素。
- 若为全白（0 黑像素），该块被舍弃（无效块），以避免产生视觉突变。

**嵌入 1：**

- 将当前块中黑色像素点数调整为 **1 个**。
- 若已有 1 个黑像素，无需修改；
- 若为 0、2 或 3 个黑像素，则**保留原黑像素，必要时将多余黑像素翻为白色**。
- 若为全黑（4 黑像素），该块被舍弃。

------

#### 信息提取规则

对嵌入后图像中的每一个 1×4 像素块：

- 黑像素数为 **1** → 提取比特 `1`；
- 黑像素数为 **3** → 提取比特 `0`；
- 黑像素数为 **0、2、4** → 视为无效块，不提取数据。



## 三、实验过程

> 本实验利用二值图像隐藏法，实现将秘密信息（文字：2211985）嵌入到位图中

**实现思想：**利用每 4 像素组成一个“数据块”，通过黑色像素数量的奇偶/数量模式来表示一个二进制比特。

##### 1. **信息表示与转换**

- **文本转二进制**：将文本字符转换为8位ASCII码（如字符'2' → `00110010`），形成二进制矩阵
- **像素块映射**：每个秘密比特（0/1）对应载体图像中4个连续像素组成的块

##### 2. **嵌入规则**

- **秘密位0的嵌入**：
  - 若当前块的黑像素数 ∈ {1,2,4} → 随机翻转2个白像素为黑
  - 若当前块的黑像素数 = 0 → 随机翻转1个白像素为黑
- **秘密位1的嵌入**：
  - 若当前块的黑像素数 ∈ {2,3} → 随机翻转2个黑像素为白
  - 若当前块的黑像素数 = 0 → 随机翻转1个黑像素为白

##### 3. **提取规则**

- 统计每个4像素块中的黑像素数量：
  - 当黑像素数 ∈ {1,3,4} → 解码为0
  - 其他情况 → 解码为1

##### 4. **空间分配策略**

- **载体图像划分**：将载体图像划分为多个4像素块，每个块承载1比特信息
- **随机化操作**：通过`randperm`实现像素翻转位置的随机选择，增强隐蔽性

#### 文本转二进制矩阵  `textToBinaryMatrix`

```matlab
function secret = textToBinaryMatrix(text)
    secret = zeros(length(text), 8);
    for i = 1:length(text)
        binStr = dec2bin(uint8(text(i)), 8); 
        secret(i,:) = binStr - '0';
    end
end
```

- 每个字符被转换为 8 位二进制，例如 `'2'` -> `00110010`
- 最终**输出为 `n×8` 的二进制矩阵**

------

#### 二进制转文本 `binaryMatrixToText`

```matlab
function decodedText = binaryMatrixToText(binaryMatrix)
    [m, ~] = size(binaryMatrix);
    decodedText = blanks(m);
    for i = 1:m
        binChar = num2str(binaryMatrix(i,:));
        binChar = binChar(binChar ~= ' ');
        decodedText(i) = char(bin2dec(binChar));
    end
end
```

- 每行是一个字符的 8 位二进制
- 转换为字符串（ASCII 解码）

------

#### 嵌入秘密信息 `hideSecret`

```matlab
function result = hideSecret(imageMatrix, numRows, numCols, secretMatrix)
    for i = 1:numRows
        for j = 1:numCols
            blackCount = countBlackPixels(imageMatrix, i, j);
            if secretMatrix(i, j) == 0
                imageMatrix = setPixelsToZero(imageMatrix, i, j, blackCount);
            else
                imageMatrix = setPixelsToOne(imageMatrix, i, j, blackCount);
            end
        end
    end
    result = imageMatrix;
end
```

- 遍历所有秘密比特位置
- 对应图像区域通过修改像素黑点数进行信息嵌入

------

#### 提取秘密信息 `extractSecret`

```matlab
function result = extractSecret(originalWithSecret, m, n)
    result = zeros(m, n);
    for i = 1:m
        for j = 1:n
            blackCount = countBlackPixels(originalWithSecret, i, j);
            result(i, j) = ~(blackCount == 1 || blackCount == 3 || blackCount == 4);
        end
    end
end
```

- 根据黑色像素数量判断隐藏的是 1 还是 0
- 黑点数为 1、3、4 表示嵌入了 0，否则为 1

------

#### 计数黑色像素 `countBlackPixels`

```matlab
function blackCount = countBlackPixels(imageMatrix, row, col) 
    [~, n] = size(imageMatrix);
    halfWidth = n / 2;
    positionInRow = (row - 1) * halfWidth + col;
    positionInMatrix = positionInRow * 4 - 3;
    matrixRow = ceil(double(positionInMatrix) / n);
    matrixCol = positionInMatrix - (matrixRow - 1) * n;

    if matrixCol + 3 > n
        blackCount = 0;
        return;
    end

    blackCount = sum(imageMatrix(matrixRow, matrixCol:(matrixCol + 3)) == 0);
end
```

- 每一个隐藏比特对应图像中连续 4 个像素位置
- 判断这 4 个像素中黑色（0）的数量

------

#### 设置 0 的像素 `setPixelsToZero`

```matlab
function result = setPixelsToZero(imageMatrix, row, col, blackCount)
    ...
    if blackCount == 1 || blackCount == 2 || blackCount == 4
        randIndices = randperm(4, 2);
        imageMatrix(matrixRow, matrixCol + randIndices - 1) = 0;
    elseif blackCount == 0
        imageMatrix(matrixRow, matrixCol + randi(4) - 1) = 0;
    end
    ...
end
```

- 为嵌入 `0`，调整像素，使黑点数为 1、3、4 中之一
- 采用随机策略选择位置设为 0

------

#### 设置 1 的像素 `setPixelsToOne`

```matlab
function result = setPixelsToOne(imageMatrix, row, col, blackCount)
    ...
    if blackCount == 0
        imageMatrix(matrixRow, matrixCol + randi(4) - 1) = 1;
    elseif blackCount == 2 || blackCount == 3
        randIndices = randperm(4, 2);
        imageMatrix(matrixRow, matrixCol + randIndices - 1) = 1;
    end
    ...
end
```

- 为嵌入 `1`，调整像素，使黑点数不是 1、3、4
- 同样采用随机策略进行修改
