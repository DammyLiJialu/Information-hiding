# 图像信息隐藏算法

### **1. 整体流程**

采用 **DCT域系数替换** 的嵌入策略，流程分为三部分：

1. **预处理阶段**：调整二值图尺寸，适配宿主图像容量
2. **嵌入阶段**：通过修改DCT中频系数嵌入信息
3. **提取阶段**：通过检测DCT系数符号提取隐藏信息

------

### **2. 核心设计步骤**

#### **2.1 预处理**

1. **宿主图像分块**
   - 将宿主图像划分为 `8×8` 像素块
   - 可嵌入块数：`bh = 宿主高度//8`, `bw = 宿主宽度//8`
2. **二值图适配**
   - 将二值图缩放至 `(bw, bh)` 尺寸（1像素对应1宿主块）
   - 二值化处理：`阈值=128`，确保像素值为 `0`（黑）或 `255`（白）
   - 使用最近邻插值保持二值性

```python
binary_image = cv2.resize(binary_image, (bw, bh), interpolation=cv2.INTER_NEAREST)
```

------

#### **2.2 信息嵌入**

1. **DCT变换**
   - 对每个 `8×8` 宿主块进行DCT变换
   - 选择中频系数 `(1,1)` 作为嵌入位置（平衡隐蔽性与鲁棒性）
2. **系数修改规则**
   - 嵌入位为 `1`：设置 `DCT(1,1) = +alpha`
   - 嵌入位为 `0`：设置 `DCT(1,1) = -alpha`
   - 通过符号差异实现信息编码
3. **逆变换与溢出处理**
   - 对修改后的DCT系数进行逆变换
   - 像素值裁剪至 `[0, 255]` 防止溢出

```python
dct_block[pos] = alpha if bit else -alpha  # 修改系数
new_block = np.clip(cv2.idct(dct_block), 0, 255)  # 逆变换+溢出处理
```

------

#### **2.3 信息提取**

1. **符号检测法**
   - 对含密图像分块进行DCT变换
   - 检测 `(1,1)` 位置系数符号：
     - `>0` → 提取 `1`（白像素）
     - `≤0` → 提取 `0`（黑像素）
2. **二值图重建**
   - 将提取的 `(bh, bw)` 二值矩阵放大至原尺寸显示

```python
extracted[i,j] = 255 if dct_block[pos] > 0 else 0  # 符号判断
extracted = cv2.resize(extracted, (bw*8, bh*8), interpolation=cv2.INTER_NEAREST)  # 显示放大
```

------

### **3. 关键参数**

| 参数    | 作用            | 典型值 | 调整影响                     |
| :------ | :-------------- | :----- | :--------------------------- |
| `alpha` | 控制嵌入强度    | 20-50  | 值越大鲁棒性↑、隐蔽性↓       |
| `(1,1)` | DCT系数嵌入位置 | 固定   | 低频区隐蔽性更好但失真风险高 |
| 块尺寸  | 分块大小        | 8×8    | 兼容JPEG标准                 |

------

### **4. 方案特点**

- **优点**：
  1. 利用DCT中频系数的感知冗余性
  2. 符号检测法无需原始图像（盲提取）
  3. 计算复杂度低（适合实时处理）
- **缺点**：
  1. 抗压缩攻击能力较弱
  2. 嵌入容量受块数量限制
  3. 需人工调整 `alpha` 平衡隐蔽性与鲁棒性

------

该方案通过 **DCT系数符号调制** 实现二值图像的隐蔽传输，核心思想是将二值信息编码为频域系数的正负极性，具有算法简单、实时性好的特点，可作为数字水印或隐蔽通信的基础框架。
