## 一、实验目的

DCT域的信息隐藏包括：

1. 修改系数方法
2. 系数比较方法。

以上两种方法任选一种，实现变换域中的信息隐藏与提取。



## 二、实验原理

#### **1. 变换域信息隐藏的核心思想**

- **抗攻击性优势**：相较于 LSB（最低有效位）在像素域直接修改低位比特的方法，变换域技术（如 DCT）将信息嵌入到频域系数中，利用人类视觉系统（HVS）对频域变化的不敏感性，使隐藏信息更难被察觉，同时抵抗常见信号处理攻击（如压缩、滤波等）。
- DCT 的特性：
  - JPEG 压缩的核心是二维 DCT 变换，图像被分割为 8×8 像素块，经 DCT 变换后得到频域系数矩阵，其中：
    - **直流（DC）系数**：左上角第一个系数，代表图像块的平均亮度，能量最高。
    - **交流（AC）系数**：其余系数，代表图像的细节和纹理。低频系数（靠近左上角）对应图像的平滑区域，高频系数（右下角）对应边缘和噪声，**中频系数（中间区域）包含图像的主要结构信息**，是 HVS 最敏感的部分。
  - **Zig-Zag 扫描**：将二维 DCT 系数转换为一维序列，按频率递增顺序排列（低频在前，高频在后），便于选择中频系数（如序列中第 2~30 个系数）进行信息嵌入。

#### **2. 中频系数的选择依据**

- **不可察觉性**：修改中频系数对图像视觉质量影响较小，因 HVS 对中频区域的微小变化不敏感。
- **鲁棒性**：中频系数在 JPEG 压缩等操作中不易被完全丢弃或大幅修改，故嵌入信息可抵抗此类攻击。



### **实验流程总结**

1. **预处理**：将载体图像分块（8×8），选择嵌入块与系数位置（基于 PRNG）。
2. **DCT 变换**：对每个块进行二维 DCT，得到频域系数矩阵。
3. **信息嵌入**：根据算法类型（叠加、比例调整或相对大小）修改中频系数。
4. **逆变换**：对修改后的系数矩阵进行 DCT 逆变换，生成含密图像。
5. **提取与检测**：对含密图像重复分块和 DCT 变换，根据嵌入算法提取信息，通过误码率（BER）评估鲁棒性，通过峰值信噪比（PSNR）评估不可察觉性。



## 三、实验过程

实现基于 DCT 变换域的信息隐藏技术，通过**修改图像的 DCT 系数**来嵌入秘密信息，并能从含密图像中准确提取。

### **核心原理**

该算法利用 DCT 变换的特性，在图像的**中频系数**中嵌入秘密信息。主要基于两个核心思想：

1. **系数相对大小关系**：通过调整特定 DCT 系数对的大小关系（如`AC(2,3) > AC(3,2)`）来表示二进制信息（1 或 0）。
2. **冗余嵌入与多数表决**：每个秘密比特用 5 对系数编码，提取时通过多数表决提高鲁棒性。



### **关键部分解析**

#### 1. **DCT 变换与分块处理**

```matlab
block = original_img(i:i+block_size-1, j:j+block_size-1);
dct_block = dct2(block);  % 对8x8图像块进行DCT变换
```

- **DCT 变换**将图像从空间域转换到频率域，低频系数集中在左上角（如 DC 系数），中频系数分布在中间区域，高频系数在右下角。
- **分块处理**：将原始图像分成 8x8 的小块，每个块独立处理，避免信息扩散。

#### 2. **秘密信息嵌入（Hide_DCT 函数）**

```matlab
% 选择5对中频系数用于嵌入1比特
coeff_pairs = [2 3; 3 2; 3 3; 2 4; 4 2];

if secret_bit == 1
    % 嵌入1: 确保val1 > val2 + ε
    val1 = val2 + 0.1;
else
    % 嵌入0: 确保val1 < val2 - ε
    val1 = val2 - 0.1;
end
```

- **系数对选择**：选择中频系数对（如`(2,3)`和`(3,2)`），因为中频对人眼不敏感，修改后不易察觉。
- 相对大小调整：
  - 嵌入`1`时，使`val1 > val2`（差值至少为 0.1）；
  - 嵌入`0`时，使`val1 < val2`（差值至少为 - 0.1）。
- **冗余嵌入**：每个秘密比特用 5 对系数编码，提高抗干扰能力。

#### 3. **秘密信息提取（Extract_DCT 函数）**

```matlab
% 提取时比较含密图像与原始图像的系数变化
if val_stego > val_ref  % 系数增大表示嵌入1
    bit_sum = bit_sum + 1;
end

% 多数表决：5对中≥3对支持1则判为1
extracted_img(secret_idx) = bit_sum >= 3;
```

- **参考系数**：提取时需要原始图像的 DCT 系数（通过`ref_coeffs`传递）作为参考，比较含密图像的系数变化方向。
- **多数表决**：5 对系数中若有≥3 对的变化方向一致，则认为该比特为 1，否则为 0。这种冗余机制大幅提高了准确率（通常 > 98%）。

#### 4. **抗干扰设计**

- **选择中频系数**：避免修改 DC 系数（对亮度影响大）和高频系数（易受噪声影响）。
- **嵌入强度控制**：参数`ε=0.1`平衡了不可见性与鲁棒性，过小易受噪声干扰，过大会降低 PSNR。
- **统计判决**：多数表决机制有效抵抗了图像压缩、加噪等干扰。



### **评估指标**

```matlab
psnr_val = psnr(uint8(stego_img), uint8(original_img));  % 图像质量评估
accuracy = sum(sum(extracted_img == secret_img)) / numel(secret_img);  % 提取准确率
```

- **PSNR（峰值信噪比）**：通常 > 35dB，表示含密图像与原始图像视觉上几乎无差异。
- **提取准确率**：通常 > 98%，表明秘密信息能被可靠恢复。



## 四、实验总结

### 【**优缺点分析**】

#### **优点**

1. **鲁棒性强**：对 JPEG 压缩、高斯噪声、滤波等攻击有较好抵抗能力。
2. **不可见性好**：修改中频系数，对图像视觉质量影响极小。
3. **高准确率**：通过冗余嵌入和多数表决，提取准确率接近 100%。

#### **缺点**

1. **需要原始图像**：提取时依赖原始图像的 DCT 系数，属于非盲提取。
2. **容量有限**：每 8x8 块仅嵌入 1 比特，适合隐藏小容量信息（如水印、ID）。

### 【**应用场景**】

- **数字水印**：在图像中嵌入版权信息或标识。
- **隐蔽通信**：在图像中传输秘密消息。
- **完整性验证**：通过提取信息验证图像是否被篡改。
