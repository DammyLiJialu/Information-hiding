2. `img_divide`函数：（将输入二值图像分割成两个视觉密码共享图 `img1` 和 `img2`）

输入图像 `img` 是二值化的黑白图像，每个像素扩展为 `2x2` 块。

`img1` 和 `img2` 采用不同的随机模式 (`randi(3)`) 生成共享图。

`img1` 和 `img2`**单独查看时无信息**，但 **叠加在一起时恢复原图像**：

- 如果原始像素是 **白色**，两个共享图的 `2×2` 块会形成相同的随机模式，使得合成后依然是白色。
- 如果原始像素是 **黑色**，两个共享图的 `2×2` 块会形成互补模式，使得合成后变成黑色。

```matlab

function [img1, img2] = img_divide(img)

    img_size = size(img);
    disp("The size of the input image is: ");
    disp(img_size);

    x = img_size(1); % x 为 img_size 的第一个参数
    y = img_size(2); % y 为 img_size 的第二个参数
    img1 = 255 * ones(2 * x, 2 * y); % 将 img1 初始化为全白图像
    img2 = 255 * ones(2 * x, 2 * y); % 将 img2 初始化为全白图像
    disp("The size of the img1 is: ");
    disp(size(img1));
    disp("The size of the img2 is: ");
    disp(size(img2));

    for i = 1 : x
        for j = 1 : y
            new_img_row = 2 * (i - 1) + 1;
            new_img_col = 2 * (j - 1) + 1;
            key = randi(3);

            switch key
                case 1
                    img2(new_img_row, new_img_col) = 0;
                    img2(new_img_row, new_img_col + 1) = 0;

                    if img(i, j) == 1 % original_img is white
                        img1(new_img_row, new_img_col + 1) = 0;
                        img1(new_img_row + 1, new_img_col + 1) = 0;
                    else % original_img is black
                        img1(new_img_row + 1, new_img_col) = 0;
                        img1(new_img_row + 1, new_img_col + 1) = 0;
                    end

                case 2
                    img2(new_img_row, new_img_col) = 0;
                    img2(new_img_row + 1, new_img_col + 1) = 0;

                    if img(i, j) == 1 % original_img is white
                        img1(new_img_row, new_img_col) = 0;
                        img1(new_img_row + 1, new_img_col) = 0;
                    else % original_img is black
                        img1(new_img_row, new_img_col + 1) = 0;
                        img1(new_img_row + 1, new_img_col) = 0;
                    end

                case 3
                    img2(new_img_row, new_img_col) = 0;
                    img2(new_img_row + 1, new_img_col) = 0;

                    if img(i, j) == 1 % original_img is white
                        img1(new_img_row, new_img_col) = 0;
                        img1(new_img_row, new_img_col + 1) = 0;
                    else % original_img is black
                        img1(new_img_row, new_img_col + 1) = 0;
                        img1(new_img_row + 1, new_img_col + 1) = 0;
                    end
            end
        end
    end
end

```

3. `img_merge`函数：（合并两张视觉密码共享图 `img1` 和 `img2`，恢复原始图像）

由于视觉密码共享图采用 **互补的方式编码黑色像素**

- 黑色部分在 `img1` 和 `img2` 中至少有一个 `0`，所以 `img_merge` 叠加后能正确恢复黑色。
- 白色部分会被分成两个相同的随机模式，叠加后仍然是白色。

```matlab

function img = img_merge(img1, img2)
    img_size = size(img1); % 两张图像的尺寸一致，故而此处以 img1 的 size 作为 img 的 size
    disp("The size of the img1 is: ");
    disp(img_size);
    
    
    x = img_size(1); % x 为 img_size 的第一个参数
    y = img_size(2); % y 为 img_size 的第二个参数
    disp("The first size of the img1 is: ");
    disp(x);
    disp("The second size of the img1 is: ");
    disp(y);

    img = 255 * ones(x, y); % 将 img 初始化为全白图像
    disp("The size of the merged img is: ");
    disp(size(img));
    
    for i = 1 : x
        for j = 1 : y
            img(i, j) = img1(i, j) & img2(i, j);
        end
    end

    disp("The size of the merged img is: ");
    disp(size(img));

end

```

2. `img_halftone`函数：

实现了 **图像半色调 (Halftoning) 处理**，使用 **Floyd-Steinberg 误差扩散算法** 将灰度图像 (`gray_img`) 转换为二值图像 (`img`)。

```matlab

function img = img_halftone(gray_img)
    img_size = size(gray_img);
    disp("The size of the gray_img is: ");
    disp(img_size);
    
    
    x = img_size(1); % x 为 img_size 的第一个参数
    y = img_size(2); % y 为 img_size 的第二个参数
    disp("The first size of the img1 is: ");
    disp(x);
    disp("The second size of the img1 is: ");
    disp(y);

    for m = 1 : x
        for n = 1 : y
            if gray_img(m, n) > 127
                out = 255;
            else
                out = 0;
            end

            error = gray_img(m, n) - out;

            if n > 1 && n < 255 && m < 255
                gray_img(m, n + 1) = gray_img(m, n + 1) + error * 7 / 16.0;  % 右方
                gray_img(m + 1, n) = gray_img(m + 1, n) + error * 5 / 16.0;  % 下方
                gray_img(m + 1, n - 1) = gray_img(m + 1, n - 1) + error * 3 / 16.0;%左下方
                gray_img(m + 1, n + 1) = gray_img(m + 1, n + 1) + error * 1 / 16.0;%右下方
                gray_img(m, n) = out;
            else
                gray_img(m, n) = out;
            end
        end
    end

    img = gray_img;

end

```

3.  `img_divide` 函数将输入的二值图像 `img` 拆分成两张图片img1，img2，而 `img_merge` 函数用于将两张分享图像合并，恢复原始图像。

```matlab

function [img1, img2] = img_divide(img)
    [x, y] = size(img);
    img1 = 255 * ones(2 * x, 2 * y); % 初始化白色图像
    img2 = 255 * ones(2 * x, 2 * y); 

    for i = 1:x
        for j = 1:y
            new_row = 2 * (i - 1) + 1;
            new_col = 2 * (j - 1) + 1;
            key = randi([0, 1]); % 生成随机模式

            if img(i, j) == 1 % 白色像素（背景）
                if key == 0
                    img1(new_row, new_col) = 0;   img1(new_row, new_col + 1) = 255;
                    img1(new_row + 1, new_col) = 255; img1(new_row + 1, new_col + 1) = 0;

                    img2(new_row, new_col) = 255; img2(new_row, new_col + 1) = 0;
                    img2(new_row + 1, new_col) = 0; img2(new_row + 1, new_col + 1) = 255;
                else
                    img1(new_row, new_col) = 255; img1(new_row, new_col + 1) = 0;
                    img1(new_row + 1, new_col) = 0; img1(new_row + 1, new_col + 1) = 255;

                    img2(new_row, new_col) = 0; img2(new_row, new_col + 1) = 255;
                    img2(new_row + 1, new_col) = 255; img2(new_row + 1, new_col + 1) = 0;
                end
            else % 黑色像素（前景）
                img1(new_row, new_col) = 0; img1(new_row, new_col + 1) = 255;
                img1(new_row + 1, new_col) = 0; img1(new_row + 1, new_col + 1) = 255;

                img2(new_row, new_col) = 0; img2(new_row, new_col + 1) = 255;
                img2(new_row + 1, new_col) = 0; img2(new_row + 1, new_col + 1) = 255;
            end
        end
    end
end


function img = img_merge(img1, img2)
    img = min(img1 + img2, 255); % 确保像素值不超过255
end
```

2. `img_halftone()` 半色调处理

   将灰度图像转换为**黑白点阵**，使用**误差扩散算法**（Floyd-Steinberg）

```matlab
function img = img_halftone(gray_img)
    [x, y] = size(gray_img);
    
    for m = 1 : x
        for n = 1 : y
            if gray_img(m, n) > 127
                out = 255; % 亮部变成白色
            else
                out = 0; % 暗部变成黑色
            end
            error = gray_img(m, n) - out;
```

通过 `if gray_img(m, n) > 127` 判断像素值是否大于 127，决定是**黑色（0）还是白色（255）**

计算误差 `error = gray_img(m, n) - out`，用于误差扩散

**误差扩散过程**

```matlab
matlabCopyEditif n < y  
    gray_img(m, n + 1) = gray_img(m, n + 1) + error * 7 / 16.0;
end
if m < x  
    gray_img(m + 1, n) = gray_img(m + 1, n) + error * 5 / 16.0;
end
if m < x && n > 1  
    gray_img(m + 1, n - 1) = gray_img(m + 1, n - 1) + error * 3 / 16.0;
end
if m < x && n < y  
    gray_img(m + 1, n + 1) = gray_img(m + 1, n + 1) + error * 1 / 16.0;
end
```

- 误差被分配到**右方、下方、左下、右下**的像素

3. `img_divide()` 视觉加密拆分

   **根据黑白像素分割**

   - **黑色像素（前景）**：两张子图像的相同位置采用**相同模式**。
   - **白色像素（背景）**：两张子图像的像素采用**互补模式**。
